Create a branded 404 page for Marvel Card Vault.

Goal:
Replace the plain/default 404 page with a fun Marvel Card Vault themed 404 page.

Creative direction:
Use the existing Marvel Card Vault logo asset. The little caped superhero from the logo should feel like the star of the page. The tone should be playful, comic-inspired, and collector-friendly.

Copy direction:
- Main badge: “404 ERROR”
- Main heading: “Oh no...”
- Subheading: “My only weakness: a 404 page.”
- Supporting copy:
  “The page you were looking for either flew off to another universe, got lost in the vault, or doesn’t exist anymore.”
- Secondary copy:
  “Try refreshing the page or head back home and keep building your collection.”
- Buttons:
  - “Back Home”
  - “Refresh”

Design direction:
- Dark comic-inspired background
- Red/black/white Marvel Card Vault color palette
- Centered polished card/container
- Use halftone/dot texture if possible
- Slight animation is welcome but not required
- Fully mobile responsive
- Keep it fun, not corporate

Must do:
- Build this as a reusable 404 page/component.
- Use the existing Marvel Card Vault logo asset from the project.
- Hook it up as the catch-all route for unknown pages.
- Preserve all existing valid routes.
- Make the Back Home button route to "/".
- Make the Refresh button reload the current page.
- Ensure mobile layout looks good.

Must not do:
- Do not change existing working routes.
- Do not change app navigation beyond adding the 404 fallback.
- Do not touch auth, billing, XP, collection logic, PC binders, image approval, Stripe, Shippo, RevenueCat, Resend, or subscription logic.
- Do not introduce Marvel/Disney affiliation language.
- Do not use copyrighted Marvel character names.
- Do not redesign the app/site globally.

Preserve:
- Existing app/site layout.
- Existing routing behavior.
- Existing Marvel Card Vault branding.
- Existing logo asset.
- Existing mobile responsiveness.

Code:

```tsx
import React from "react";
import "./not-found.css";

// Update this import path to match where the logo lives in this project.
// Use the existing Marvel Card Vault logo asset.
import logo from "../assets/marvel-card-vault-logo.png";

export default function NotFound() {
  const goHome = () => {
    window.location.href = "/";
  };

  const refreshPage = () => {
    window.location.reload();
  };

  return (
    <main className="not-found-page">
      <div className="nf-background-pattern" />

      <section className="not-found-card" aria-labelledby="not-found-title">
        <div className="nf-badge">404 ERROR</div>

        <div className="nf-logo-wrap">
          <img
            src={logo}
            alt="Marvel Card Vault"
            className="nf-logo"
          />
        </div>

        <h1 id="not-found-title" className="nf-title">
          Oh no...
        </h1>

        <h2 className="nf-subtitle">
          My only weakness: a 404 page.
        </h2>

        <p className="nf-copy">
          The page you were looking for either flew off to another universe,
          got lost in the vault, or doesn’t exist anymore.
        </p>

        <p className="nf-copy nf-copy-secondary">
          Try refreshing the page or head back home and keep building your collection.
        </p>

        <div className="nf-actions">
          <button
            type="button"
            className="nf-button nf-button-primary"
            onClick={goHome}
          >
            Back Home
          </button>

          <button
            type="button"
            className="nf-button nf-button-secondary"
            onClick={refreshPage}
          >
            Refresh
          </button>
        </div>
      </section>
    </main>
  );
}

import { Route, Switch } from "wouter";
import NotFound from "./pages/NotFound";

<Switch>
  {/* existing routes stay above this */}

    <NotFound />
  </Route>
</Switch>
